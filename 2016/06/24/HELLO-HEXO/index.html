初めまして、HEXO君、私はニャンコちゃんです。どうぞよろしくお願いします∠( ᐛ 」∠)＿
hexo圈萌♥，亲手搭窝，一本满足。


今天真是说不粗的嗨森呢(～￣▽￣)～*
真的要好好感谢阿成33的先见之明和铺路工作，提前教了我sublime text（好吧我到底学会了什么T皿T），蠢货如我也能搭个博客粗来了~~~(●’ω’●)

诶豆······事情的起源是这样的~~昨晚熬夜补高酥作业又抢课抢到凌晨两三点还被蚊子嗡嗡嗡嗡地折磨了半夜，第二天的汇编当然是理所当然的不会编辣，C艹也憋不出几行代码23333，于是我就自暴自弃地跑去开小差了（逃··· ( •̀ ω •́ )

本来想啊快期末了我还是不要作死好好复习吧，但在机（lao）缘（shi）巧（cui）合（cu）下最后还是拾起来了而且还越来越兴奋【excited(★ ω ★)】

做的时候google了一堆教程，步骤什么的都很清晰、、不过本智障紧张地摸索着也就没精力截图记录过程什么的，在这里贴出对我帮助最大的两篇博客，蒸的是、包教包会啊！

★史上最详细的Hexo博客搭建图文教程 --by Xuanwo https://xuanwo.org/2015/03/26/hexo-intor/
★成功叛逃到hexo --by lechain （伪征婚帖o(*￣▽￣*)ブ）http://old.lyeec.me/jekyll-to-hexo/

我是在Windows下搭的博客，大致流程是这样的~~

①首先下载node.js，保持默认一路Next就好了
    戳这里：https://nodejs.org/dist/v4.4.5/node-v4.4.5-x64.msi 
②配置好node.js环境之后，到github（https://github.com/）上申请一个帐号 
③接下来完成git的安装和配置
④使用GitHub Pages解锁各种新姿势=v=
    看使用帮助说不定能更好的理解GitHub Pages是做什么用的：
    https://help.github.com/categories/github-pages-basics/
⑤······

写流程是帮助自己回忆吧，具体的操作大神们的教程里讲的比我清楚多了，嘿嘿（逃···）

这个时候已经可以用hexo s命令预览你的博客了，它会给出这样的提示：Hexo is running at http://localhost:4000/. 然后······你会一边预览一边发现······嗯、有点丑。

所以我们需要一个主题( •̀ ω •́ )y

主题的话，我直接吃了一枚安利：https://github.com/MOxFIVE/yelee
作者给出了超级详细的使用说明~~！http://moxfive.coding.me/yelee/

总之自己还是智障了、在更改Author的时候（总不能用人家默认的名字吧···）居然产生了一个不大不小的错误、、没法hexo s了（哭唧唧···）它说错误出现在repo那行代码第七列,瞬间生无可恋这要上哪找去、、（最后才知道这是在提示我_config.yml中deploy那一步我冒号后面没加空格>o<~）最后的最后，我在更改完主页的标题个人信息并保存之后，重新hexo g了一下，重新上传自己的博客。好的这次就成功了~~

到这基本上就差不多了（？····）好吧这完全是我的错觉（摔！）（╯－＿－）╯╧╧

本以为按照教程上说的generate一下再deploy一下输个github的账号密码就OK了可素没想到啊他他他居然失败了啊T皿T

Permission denied (publickey).
fatal: Could not read from remote repository.
Please make sure you have the correct access rights and the repository exists.

表示一脸蒙蔽赶紧灰溜溜的跑去谷歌，发现好多人和我有一样的问题、、最后大家讨论出来的解决方案居然是更换一个较低版本的git······orz
可是我懒啊······
于是我想不如改用sshkey，如果成功了的话好像会更方便一些（虽然不懂其中原理）~~
锵锵~！干货教程在此 http://old.lyeec.me/git/ 阿成33的博客真是个好地方吖~~~∠( ᐛ 」∠)＿

然后就···然后就成功惹~！✧٩(ˊωˋ*)و✧

诶诶诶不行了我已经做不到一本正经地胡说（zhuang）八道（bi）了，明明别人家的博客都是干货我这怎么全是水呢 (～￣▽￣)→))￣▽￣)o天哪难道我只能靠强行卖萌来留住人气了么sad、

就个人感觉而言，这种类型的博客是私人性质比较高的一种，莫名喜欢(❤ ω ❤)，在秋秋上自己发的东西大家都能看到，我自己是内心活动比较丰富的人，如果表现的太频繁了的话难免会被认为有装X的嫌疑，微信虽然好友少了点，可那个圈子太无聊了没意思，一直想有一个自己的小天地啊啊啊，可以毫无顾忌地说自己喜欢的话~~天天自嗨天天水（怎么感觉有种注孤生的辛酸=……=）

本来还想买域名的···DEMO好贵···改天吧、改天再买QAQ（改天是哪天啊摔！）

可怕的是都这么晚了我居然不会困诶〒_〒果然自己探索喜欢的东西什么的真的好燃o(￣▽￣)ブ比复习高酥大雾有意思多了啊~demo貌似吾辈真的要挂科了呢~~如果可以早早起床的话我就熬粥喝嗷嗷嗷想想就鸡冻的流下了口水（❤ ω ❤）

呐呐，Private Castle就是内心的具象化吧~哇唔~发明出这么美好的东西的人真是太赞了~(★ ω ★)我会像照顾恋人一样照顾好我的博客（虽然并不知道恋人这种生物要怎么照顾）不过我想给它100%的care应该就够了吧~(～￣▽￣)～*

深刻地意识到自己弱到爆炸，还有hin多姿势要学，不过吾辈不怕被嫌弃~欢迎大家常来玩耍~~ (～￣▽￣)→))￣▽￣)o

最后的最后还要再感谢一次阿成33，33赛高！！✧٩(ˊωˋ*)و✧
