第六话：IO

File类

File类被定义为“文件和目录路径名的抽象表示形式”，这是因为File类既可以表示“文件”也可以表示“目录”，他们都是通过对应的路径来进行描述的。通过构造函数创建一个File类对象，则该对象就是指定文件的引用，可以通过该对象对文件操作。
例如：File f = new File(“d:\temp.txt”);
创建了一个File类对象f,可通过f对temp.txt进行操作。

File类方法

但是，File类中没有提供对文件的读写操作，如果要实现对文件的读写，需要通过io
流来进行操作。

IO流的概念

IO流就是以字节序或字符序对文件进行读取或写入，相当于在读取或写入前给文件添加一个管道，数据通过管道进行传输。

以下是stackoverflow上的一个问题“什么是IO流”对应的答案：

字节流

单位：8位，通常用于文件上传下载，网络信息传递等。
实现对文件的读写，主要用到FileInputStream和FileOutputStream。是InputStream和OutputStream（抽象类）对应的子类。

文件读写步骤

  1. 创建文件对象：File f = new File(“文件路径”);
  2. 根据文件对象创建输入输出流
    • FileInputStream filein = new FileInputStream(f);
    • FileOutputStream fileout = new FileOutputStream(f);
  3. 实现对文件的读写
    • 读：filein.read(字节数组);//将文件内容读到指定的字节数组
    • 写：fileout.write(字节数组);//将指定字节数组中的数据写到文件
  4. 关闭流
    • filein.close();
    • fileout.close();
