深入理解Python密码库Cryptography——AES五种模式（ECB/CBC/CFB/OFB/CTR）

  • AES
  • Cryptography
  • Python
  • 密码学

Cryptography，了解一下？_(:3」∠❀)_

开这篇主要是为了贴代码的wwwww。
Python密码库Cryptography里有很多与AES加密相关的东西。
使用度最广的应该是Fernet，这个是对新手友好的一个加密模块，底层使用了AES+CBC模式
其他模式需要自己实现，但其实也不难，主要参数是modes
写好第一个的注释以后，其他四个都是复制粘贴，哈哈~

加解密的过程大同小异
首先处理密钥，然后用Cipher类初始化一个加密实例，这里要选好加密算法和模式，然后调用相关函数就可以加密解密辣。

关于密钥，可以使用os.urandom(n)直接生成，如果要用PBKDF2HMAC处理（对应代码里的kdf），注意解密时需要重新初始化一次kdf
关于模式，ECB不需要初始向量，CTR需要一个nonce，其他的都需要初始向量iv

每个模式大概要50行代码，各模式间有很多相似的地方，具体使用参见main函数
