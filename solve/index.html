并查集是维护“同一组”的数据结构，那么对每个动物创建三个元素i−A, i−B, i−C。
建立3×N大小的并查集。
i−x表示i属于x。
并查集同一组的所有元素表示的情况都同时发生。
由于每个动物所在的种类是不确定的，所以当给出消息判断的时候，需要考虑 i在A,B,C三种情况，那么将三种情况都合并起来即可减少判断量。即
情况一
Union(x, y); // 当x和y都在A组
Union(x+N, y+N); // 当x和y都在B组
Union(x+2N, y+2N); // 当x和y都在C组
情况二
Union(x, y+N); // 表示A的x吃B的y
Union(x+N, y+2N); // 表示B的x吃C的y
Union(x+2N, y); // 表示C的x吃A的y
